# GGame Launcher 🎮

Um launcher minimalista e elegante para desktop, construído com Flutter e focado em imersão.

## Funcionalidades
* Interface focada em arte (Hero Banners e Efeitos de Blur).
* Diário de Jogos integrado (Notas, Hype Meter e Geladeira).
* Navegação 100% nativa por controle (Xbox/PlayStation).
